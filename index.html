Planar parallel robotics

Five-bar linkage design tools

Interactive tools for exploring five-bar linkage geometry, reachability, singularities, and idealised holding torque for a planar parallel mechanism with a rigid L5 tool extension.

Open simulator View singularity plot GitHub

What the mechanism is

The model is a two-actuator planar five-bar linkage. Two base joints drive proximal links, passive elbows close the loop at a wrist point, and L5 extends one distal link to form the tool tip.

What the simulator shows

The simulator shows joint control, polar target inverse kinematics, fixed IK branch pairs, branch-switch prevention, trajectory traces, singularity margins, and quasi-static holding torque for a tip load.

Condition number

The condition number is the ratio of the largest to smallest singular value of the linkage Jacobian. Values near 1 are well conditioned; high values indicate weak or poorly controlled motion directions.

Serial singularity

A serial singularity occurs when one side chain is folded or stretched so that its actuator loses useful leverage over the wrist or tool point.

Parallel singularity

A parallel singularity occurs when the closed-chain constraints become locally dependent. Near this state, the mechanism can lose stiffness or gain an uncontrolled direction.

Limitations

These are ideal kinematic and quasi-static tools. They do not model link mass, friction, backlash, compliance, cable stretch, dynamics, actuator limits, or controller behaviour.

Current best practical config

The current default short-link configuration uses the fixed (+1, -1) branch and reaches the sampled 1 m upper semicircle with max condition around 6.61.
